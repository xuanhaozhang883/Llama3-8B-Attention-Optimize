# Historical RoPE Outputs

The files in this directory were generated by an earlier RoPE experiment and do not have their matching input vectors in the current repository. They are kept for reference only.

They are not used by:

- `RoPE/tb_rope_qk_file.sv`
- `RoPE/tools/prepare_rope_vectors.py`
- `RoPE/run_rope_sim.tcl`

Do not use `rope_compare_result.txt` as evidence for the current `golden_model_outputs/fpga_slice` verification. The current test flow uses `RoPE/data/` and reports the actual BF16 mismatch against the shared Golden data.
